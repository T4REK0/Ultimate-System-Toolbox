ULTIMATE SYSTEM TOOLBOX - دليل التشغيل السريع
================================================

📌 ملف واحد فقط: toolbox.py
👨‍💻 يحتوي على 32 أداة احترافية

================================================
🚀 طريقة التشغيل (Windows)
================================================

الخطوة 1: فتح موجه الأوامر (CMD)
---------------------------------
اضغط على Windows + R 
اكتب cmd ثم Enter

أو
ابحث عن "Command Prompt" في قائمة ابدأ

الخطوة 2: الذهاب إلى مكان الملف
--------------------------------
استخدم الأمر cd للذهاب إلى المجلد الذي يحتوي على الملف

مثال:
cd C:\Users\YourName\Desktop

(غير YourName باسم المستخدم الخاص بك)

الخطوة 3: تشغيل البرنامج
------------------------
اكتب الأمر التالي:

python toolbox.py

إذا لم يعمل، جرب:

py toolbox.py

أو

python3 toolbox.py

الخطوة 4: انتظر قليلاً
-----------------------
- سيبحث البرنامج عن منفذ متاح
- سيقوم بتشغيل خادم محلي
- سيفتح المتصفح تلقائياً

إذا لم يفتح المتصفح تلقائياً:
افتح المتصفح واكتب:
http://localhost:8080

================================================
🖥️ طريقة التشغيل (Mac / Linux)
================================================

1. افتح Terminal
2. انتقل إلى مكان الملف:
   cd /path/to/your/file

3. شغل البرنامج:
   python3 toolbox.py
